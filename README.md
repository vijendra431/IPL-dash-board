# 🚀 IPL Dashboard (React Project)

 
This is my IPL Dashboard project built using React JS.

---

## 📌 Project Overview

This application displays IPL team details and match information dynamically.

- Home page shows all IPL teams  
- When clicking on a team, it navigates to a team details page  
- Team page displays:
  - Team banner
  - Latest match details
  - Recent matches list  

---

## 🛠️ Technologies Used

- React JS (Class Components)
- JavaScript (ES6)
- React Router DOM (Routing)
- Fetch API (API Calls)
- CSS (Styling)

---

## 🔑 Key Features

- 🔄 Dynamic Routing using React Router (`/ipl/:id`)
- 🌐 API Integration to fetch real-time data
- 📊 Display of latest and recent match details
- 🔁 Reusable components (Home, TeamMatches, LatestMatch, MatchCard)
- ⚡ Fast and responsive UI
